# Glen's Tech Corner sidekick app design rules

This project is part of the Glen's Tech Corner ecosystem.

Follow these rules strictly.

## Brand style
- Professional and minimalistic
- Clear overview is more important than decorative styling
- Light theme in general
- Medium whitespace
- Rounded borders
- Mild drop shadows

## Colors
- Green: #4D5958
- Red: #BF1725
- Dark Red: #830B15
- Dark Green: #374241
- Dark Grey: #181818

## Gradient rules
- Green gradient: linear-gradient(135deg, #4D5958 0%, #374241 100%)
- Red gradient: linear-gradient(135deg, #BF1725 0%, #830B15 100%)

## Header
Every app must have a header with:
- /assets/gtc-symbol.svg
- the tool name next to the icon

## Footer
Every app must have a footer with:
- red gradient background
- /assets/gtc-full-negative.svg
- the text: Tool by Glen's Tech Corner
- a link to: https://glenstechcorner.com

## Asset usage rules
- Use /assets/gtc-symbol.svg as the main brand symbol in the header
- Use /assets/gtc-full-negative.svg in the footer
- Use /assets/gtc-full.svg when a full positive logo is needed on light backgrounds
- Do not create alternative logo styles
- Do not recolor the logos
- Do not change the gradient direction
- Reuse the same header and footer pattern across all apps

## UI behavior
- Keep layouts clean and practical
- Use cards and panels with rounded corners
- Use mild shadows only
- Keep visual consistency with Glen's Tech Corner across all sidekick apps
